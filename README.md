# SellSmart

A modern, platform-centric web application empowering micro-businesses to establish an online presence and connect directly with their local customers. Built using React and Vite.

## 🚀 Setup & Installation

To run this project locally, make sure you have [Node.js](https://nodejs.org/) installed on your machine.

**1. Clone the repository**
```bash
git clone https://github.com/your-username/SellsSmart.git
cd SellsSmart
```

**2. Install dependencies**
```bash
npm install
```

**3. Start the development server**
```bash
npm run dev
```

The application will now be running on your local machine, usually accessible at `http://localhost:5173`.
